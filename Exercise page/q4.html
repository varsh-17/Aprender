Write a javascript program that accepts the users first name and last name of the user and generates an email with the first three characters of first name and last three characters of last name.

Copied!

Solution:

<!DOCTYPE html> <head> <title>Email</title> </head> <body style = "margin: 50px;"> <p>Enter first name : </p> <input type = "text" id="fname"><br><br> <p>Enter last name : </p> <input type = "text" id="lname"><br><br> <button onclick="email()">Generate Email</button> <p id = "gm"></p> <script> function email(){ var fname = document.getElementById("fname").value; var lname = document.getElementById("lname").value; let part1 = fname.substring(0,3); let part2 = lname.slice(-3); let gmail = part1+"."+part2+"@gmail.com"; document.getElementById("gm").innerHTML = gmail; } </script> </body>
