Find the even numbers in two arrays and concat them into a third array.

Copied!

Solution:

<DOCTYPE html> <head> <title>Even elements</title> </head> <body> <h3>Finding Even numbers : </h3> <p>Array 1 : </p><p id = "arr1"></p> <p>Array 2 : </p><p id = "arr2"></p> <p>The even elements in arrays are : </p> <p id = "ans"></p> <script> var arr1 = [1,2,3,4,5]; var arr2 = [6,7,8,9,10]; var even = []; var even1 = arr1.filter(number => number % 2 == 0); var even2 = arr2.filter(number => number % 2 == 0); even = even1.concat(even2); document.getElementById("arr1").innerHTML = arr1.join(","); document.getElementById("arr2").innerHTML = arr2.join(","); document.getElementById("ans").innerHTML = even.join(","); </script> </body>
