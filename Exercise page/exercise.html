Write a JavaScript code to display a message when a caps lock and alt key are pressed.

Copied!

Solution:

<!DOCTYPE html> <head> <title>ALT & CAPSLOCK</title> <style> #text {display:none;color:red} </style> </head> <body bgcolor = "#E1EBEE"> <h2>Press a key on the keyboard in the input field to find out if the ALT key was pressed or not.</h2> <input type="text" onkeydown="isKeyPressed(event)" value = "Input field"> <p id="ans"></p> <h2>Press the "Caps Lock" key inside the input field.</h2> <input id="myInput" value = "Input field"> <p id="text">WARNING! Caps lock is ON.</p> <script> function isKeyPressed(event) { var x = document.getElementById("ans"); if (event.altKey) { x.innerHTML = "The ALT key was pressed!"; } else { x.innerHTML = "The ALT key was NOT pressed!"; } } var input = document.getElementById("myInput"); var text = document.getElementById("text"); input.addEventListener("keyup", function(event) { if (event.getModifierState("CapsLock")) { text.style.display = "block"; } else { text.style.display = "none" } }); </script> </body> </html>
